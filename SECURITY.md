# Security Policy

## Supported versions

This project is currently in `v0`, and only the latest `main` branch is
considered supported for security fixes.
For compatibility policy context, see `README.md`.

## Reporting a vulnerability

Please report vulnerabilities privately using GitHub Security Advisories:

- <https://github.com/Warashi/go-graft/security/advisories/new>

If that channel is unavailable, contact the repository owner via:

- <https://github.com/Warashi>

Please include as much detail as possible:

- Affected version or commit
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation (if available)
